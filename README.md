# Solar Browser Wallet

This project was created by leitesv [leitesv.dev](https://leitesv.dev/) (leitesv@tuta.io).

## Installation

In the project directory, you can run:

### `yarn install`

And then:

### `yarn postinstall`

## Build 

### `yarn build`

## Test on Firefox 

### `yarn start:firefox`
